## About The Game

Welcome to The Love Kingdom 2.0. This is an updated visual novel of the original one!

## Features

- Story-driven informative visual novel experience
- Choice-based dialogue moments
- Custom GUI and themed assets
- Built with Ren'Py

## Run Locally

1. Install Ren'Py from <https://www.renpy.org/>
2. Open the Ren'Py launcher
3. Add this project folder
4. Launch the project

## Project Structure

```text
game/
	script.rpy      # Main story script
	screens.rpy     # UI screens and menus
	gui.rpy         # GUI setup
	options.rpy     # Project configuration
	audio/          # Music and SFX
	images/         # Sprites/backgrounds
```

## Development Notes

I am still learning, and this game is part of that journey. Feedback is always welcome and helps me improve.

## Troubleshooting

- If the game does not start, make sure your Ren'Py version is up to date.
- If assets are missing, confirm folders in `game/` were copied correctly.
- If you see script errors, open `traceback.txt` for details.
